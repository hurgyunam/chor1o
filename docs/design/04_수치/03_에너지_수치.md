# 에너지 수치

이 문서는 멤버 에너지의 최대치, 소모량, 회복량 및 체력 스탯과의 연동 방식을 정의한다.  
에너지 개념 정의는 [멤버 요소](../02_개념/02_멤버_요소.md)를 참조한다.

---

## 1. 에너지 최대치

에너지는 스탯이 아닌 **소모·회복되는 자원**이다.  
멤버의 **체력 스탯**이 에너지 최대치와 소모 내성을 결정한다.

```
에너지 최대치 = 50 + (체력 × 0.5)
```

| 체력 스탯 | 에너지 최대치 |
|----------|-------------|
| 10 | 55 |
| 30 | 65 |
| 50 | 75 |
| 70 | 85 |
| 90 | 95 |
| 100 | 100 |

- 에너지는 항상 **0 ~ 최대치** 사이의 정수를 유지한다
- 에너지가 0이 된 멤버는 해당 루프 내 **추가 활동 불가** 상태가 된다

## 2. 에너지 소모

### 활동별 기본 소모량

| 활동 | 기본 에너지 소모량 |
|------|------------------|
| 노래연습 (1세션) | 8 ~ 15 |
| 안무연습 (1세션) | 12 ~ 20 |
| 동시 연습 (노래+안무) | 18 ~ 30 |
| 녹음 (1곡 기준) | 10 ~ 18 |
| 무대 퍼포먼스 (1회) | 20 ~ 35 |
| 상담 (1회) | 3 ~ 6 |
| 팬미팅 (1회) | 8 ~ 14 |
| 콘서트 (1회) | 30 ~ 50 |

### 파트 블럭 소모량 연동

파트 분배 완료 시, 해당 파트 블럭의 **에너지 소모량**이 결정된다.  
파트 길이와 안무 강도(장르 기반)에 따라 소모량이 달라진다.

```
파트 블럭 에너지 소모량 = 기본값 × 파트 길이 보정 × 안무 강도 보정
```

**안무 장르별 강도 보정 계수**

| 안무 장르 | 강도 보정 계수 |
|----------|--------------|
| 청순 | 0.7 |
| 팝 | 0.85 |
| 힙합 | 1.0 |
| 걸크러시 | 1.15 |
| 퍼포먼스 | 1.3 |
| 가창없음 (안무만) | 안무 장르 계수 × 1.1 |

- 기본값: **10**
- 파트 길이 보정: `(파트 길이(초) / 15초)^0.7`
- 파트 블럭 소모량의 합이 해당 곡의 **멤버별 에너지 소모량**이 된다

### 체력에 따른 소모 내성

체력 스탯이 낮을수록 동일 소모량에서 퍼포먼스 저하가 더 빠르게 발생한다.

```
실질 에너지 소모량 = 기본 소모량 × 내성 계수
내성 계수 = 1.5 - (체력 / 100) × 0.5    (범위: 1.0 ~ 1.5)
```

| 체력 스탯 | 내성 계수 | 실질 소모 (기본 20 기준) |
|----------|----------|------------------------|
| 10 | 1.45 | ≈ 29 |
| 30 | 1.35 | ≈ 27 |
| 50 | 1.25 | 25 |
| 70 | 1.15 | 23 |
| 90 | 1.05 | ≈ 21 |
| 100 | 1.0 | 20 |

## 3. 에너지 회복

### 회복 수단별 회복량

| 회복 수단 | 회복량 |
|----------|--------|
| 휴식 (1일) | 15 ~ 25 |
| 집중 휴식 (1일, 창작 활동 전면 중단) | 25 ~ 40 |
| 상담 후 휴식 | 추가 10 (상담 효과와 복합 적용) |

- 에너지 회복 시 **멘탈 상태**에 따라 회복 효율이 달라진다

**멘탈 상태에 따른 회복 효율**

| 멘탈 스탯 | 회복 효율 |
|----------|----------|
| 1 ~ 20 | × 0.5 (탈진 상태, 회복이 거의 안 됨) |
| 21 ~ 40 | × 0.7 |
| 41 ~ 60 | × 1.0 (기준) |
| 61 ~ 80 | × 1.15 |
| 81 ~ 100 | × 1.3 |

## 4. 에너지와 퍼포먼스 저하

에너지가 부족한 상태에서 활동을 강행하면 퍼포먼스 저하가 발생한다.  
이는 **소화 밸류에 패널티**를 적용하는 형태로 반영된다.

```
에너지 부족 패널티 = max(0, (에너지 적정 기준 - 현재 에너지) / 에너지 적정 기준) × 0.4
최종 소화 밸류 = 원래 소화 밸류 × (1 - 에너지 부족 패널티)
```

- **에너지 적정 기준**: `에너지 최대치 × 0.3`  
  (최대치의 30% 이상 남아있으면 패널티 없음)
- 에너지 0일 때 최대 패널티 40% 적용 → 소화 밸류가 최대 60% 수준까지 하락할 수 있다

**에너지 상태별 퍼포먼스 요약**

| 에너지 상태 | 소화 밸류 보정 |
|------------|--------------|
| 적정 기준 이상 | 없음 (정상) |
| 적정 기준 미만 ~ 10% | × 0.85 ~ 0.95 |
| 10% 미만 ~ 1 | × 0.65 ~ 0.85 |
| 0 (완전 소진) | 활동 불가 |

---

## 5. 설계 의도 요약

- 에너지는 **루프 내 자원 배분**의 핵심 압박 요소다
- 연습을 많이 할수록 스탯이 오르지만 에너지가 줄어든다 → 연습량과 컨디션 사이의 트레이드오프
- 체력이 낮은 멤버에게 에너지 소모가 큰 파트를 배정하면 퍼포먼스 저하로 이어진다 → 파트 배정의 전략적 의미 강화
- 에너지 0 상태의 멤버가 생기지 않도록 관리하는 것이 플레이어의 핵심 과제 중 하나다
