# 04_수치

이 폴더는 게임 내 모든 수치 체계를 정의한다.
02_개념과 03_창작루프에서 정의한 개념들의 **구체적 수치, 범위, 계산 공식**을 확정한다.

---

## 문서 목록

```
00_수치_체계_개요.md    수치 설계 철학, 전체 수치 구조 개요
01_멤버_스탯_수치.md    스탯 범위·성장·소화 밸류 계산
02_밸류_계산_공식.md    파트 밸류·곡 밸류·앨범 밸류 산출 공식
03_에너지_수치.md       에너지 소모·회복 수치 및 체력 연동
```

## 이 폴더의 전제

- 모든 수치는 **플레이어에게 직접 노출되지 않는다** — 정보 노출 원칙(메타_요소 참조)을 따른다
- 수치는 설계자가 내부적으로 확정하기 위한 것이며, UI에서는 근사치·힌트 형태로 표현된다
- 이 폴더의 수치는 **초안 값**으로, 실제 밸런스 조정은 별도의 밸런스 테스트 폴더에서 진행된다
- 이 폴더 내 문서는 최신화되지 않으며, 수치 변경 이력은 밸런스 테스트 폴더를 기준으로 관리한다
