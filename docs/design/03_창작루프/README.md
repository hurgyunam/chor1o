# 03_창작루프

이 폴더는 게임의 핵심 반복 구조인 **창작 루프**의 전체 흐름을 설계하는 공간이다.
`02_개념`에서 정의한 요소들이 실제로 어떤 순서로, 어떤 조건에서 연결되는지를 기술한다.

---

## 문서 목록

### `00_루프_전체_흐름.md` — 창작 루프 전체 흐름도
한 사이클의 창작 루프가 시작부터 끝까지 어떻게 전개되는지를 개괄적으로 기술한다.
돌발 서사가 루프 어느 시점에 개입할 수 있는지도 포함한다.

### `01_곡_제작_단계.md` — 곡 제작 단계별 상세 흐름
작곡 의뢰부터 녹음 완료까지, 각 단계에서 플레이어가 내리는 결정과 대기 구조를 정의한다.

### `02_돌발_서사_개입.md` — 돌발 서사 개입 구조
창작 루프의 각 단계에서 돌발 사건이 끼어드는 시점, 방식, 플레이어의 대응 흐름을 정의한다.

### `03_앨범_출시.md` — 앨범 기획 및 출시 프로세스
녹음완료 이후 앨범을 기획하고 공개하는 절차를 정의한다.
