# 창작 루프 전체 흐름

이 문서는 한 사이클의 창작 루프가 시작부터 끝까지 어떻게 전개되는지를 개괄적으로 기술한다.
각 단계의 상세 내용은 개별 문서를 참조한다.

---

## 루프 개요

창작 루프는 **곡 제작(반복) → 앨범 출시 → 공개 후 활동**의 반복 사이클이다.
루프 1회는 하나의 앨범을 완성하고 공개하는 과정 전체를 의미한다.
앨범 하나에는 **여러 곡이 수록될 수 있으며**, 곡 제작(PHASE 1~4)은 수록곡 수만큼 반복된다.
곡이 공개된 이후에는 안무 의뢰 및 무대 활동 등 **후속 활동**을 진행할 수 있다.
루프가 반복될수록 멤버가 성장하고, 팬덤이 쌓이며, 더 높은 난이도의 창작 선택이 가능해진다.

---

## 전체 흐름도

각 PHASE는 내부 작업이 끝난 뒤 반드시 **돌발 체크포인트**를 거친다.
체크포인트에서 돌발 사건이 발생하면 해당 PHASE 내에서 재조정 후 다음 PHASE로 넘어간다.

곡 제작(PHASE 1~3)은 **앨범에 담을 곡 수만큼 반복**된다. 모든 곡 제작이 완료된 뒤 앨범 출시(PHASE 4~5)로 진입한다.
뮤직비디오(MV)는 곡마다 개별적으로 의뢰할 수 있으며, **출시 전(곡 제작 중)** 또는 **출시 후(후속 활동)** 어느 시점에든 의뢰 가능하다.
안무 의뢰, MV 후공개, 무대 활동은 **곡 공개 이후** 후속 활동으로 진행된다.

```
[ 루프 시작 ]
      │
      ▼
┌─────────────────────────────────────────────────────┐
│  PHASE 1: 작곡 의뢰                                  │
│  - 작곡팀 선택 → 제안 수락 → 대기 → 곡(파트 블럭) 수령  │
│                                                     │
│  ▶ 체크포인트: 계약 파기 / 돌발 결과물 납품             │
│    → 재선택 또는 수락 후 PHASE 2 진입                  │
└──────────────────────┬──────────────────────────────┘
                       │ 곡 상태: 미완성 → 작곡완성
                       ▼
┌─────────────────────────────────────────────────────┐
│  PHASE 2: 파트 분배                                  │
│  - 각 파트 블럭에 멤버 배정                           │
│                                                     │
│  ▶ 체크포인트: 멤버 파트 불만 / 멤버 간 갈등            │
│    → 파트 재조정 또는 상담 후 분배 확정                 │
└──────────────────────┬──────────────────────────────┘
                       │ 곡 상태: → 파트분배완료
                       ▼
┌─────────────────────────────────────────────────────┐
│  PHASE 3: 녹음 및 출시 전 MV 의뢰 (선택)              │
│  - 멤버 노래연습 (가창 스탯 vs 파트 요구 레벨)          │
│  - 녹음 실행                                         │
│  - 이 곡의 MV 출시 전 의뢰 여부 결정 (선택)            │
│    → 의뢰 시: MV팀 선택 → 제안 수락 → 대기 → MV 수령   │
│                                                     │
│  ▶ 체크포인트: 멘탈 붕괴 / 멤버 부상 / MV 촬영 사고     │
│    → 파트 재배정·휴식 후 녹음 속행 또는 MV 없이 강행     │
└──────────────────────┬──────────────────────────────┘
                       │ 곡 상태: → 녹음완료 (MV 의뢰 시 → MV완성)
                       ▼
               ┌───────────────┐
               │ 수록곡 추가?   │
               └───────┬───────┘
              예 ◀─────┘──────▶ 아니오
              │                      │
              ▼                      ▼
   [ PHASE 1로 돌아가 다음 곡 제작 ]  │
                                     ▼
┌─────────────────────────────────────────────────────┐
│  PHASE 4: 앨범 기획 및 출시 준비                      │
│  - 앨범 구성 결정 (타이틀곡, 수록곡 배치)              │
│  - 출시 방식 결정 (음원 등록 / 피지컬 앨범)             │
│                                                     │
│  ▶ 체크포인트: 루머·스캔들                            │
│    → 출시 연기 또는 강행                              │
└──────────────────────┬──────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────┐
│  PHASE 5: 공개 및 성과 확인                          │
│  - 앨범 공개                                         │
│  - 인지도·팬덤 반응 확인                              │
│  - 멤버 에너지·멘탈 상태 점검                          │
│                                                     │
│  ▶ 체크포인트: 취향 이탈 / 예상 밖 대박 / 이탈 위협     │
│    → 다음 루프 방향성 힌트로 반영                      │
└──────────────────────┬──────────────────────────────┘
                       │ 앨범 공개 완료
                       ▼
┌─────────────────────────────────────────────────────┐
│  PHASE 6: 후속 활동 (선택)                           │
│  - MV 후공개 의뢰: 곡별로 MV팀 선택 → 수락 → MV 수령  │
│  - 안무 의뢰: 안무팀 선택 → 수락 → 안무 수령           │
│  - 멤버 안무 연습 및 무대 퍼포먼스 준비                 │
│  - 음방·팬미팅·콘서트 등 무대 활동                     │
│                                                     │
│  ▶ 체크포인트: 작업 지연 / 무대 사고 / 멤버 컨디션 이슈  │
│    → 일정 조정 또는 활동 축소 후 종료                   │
└──────────────────────┬──────────────────────────────┘
                       │
                       ▼
                [ 다음 루프 시작 ]
```

## 돌발 체크포인트 원칙

돌발 사건은 각 PHASE 작업이 마무리되는 시점, 즉 **PHASE 내 체크포인트**에서 처리된다.
PHASE를 가로지르는 역행은 발생하지 않는다. 재조정이 필요한 경우에도 **현재 PHASE 내에서** 해결하고 다음 단계로 넘어간다.

| PHASE | 체크포인트 발생 시점 | 처리 범위 |
|-------|---------------------|-----------|
| PHASE 1 (작곡 의뢰) | 곡 수령 직후 | 재선택 또는 결과물 수락 — PHASE 1 내 완결 |
| PHASE 2 (파트 분배) | 분배 확정 직전 | 파트 재조정 또는 상담 — PHASE 2 내 완결 |
| PHASE 3 (녹음·MV 의뢰) | 녹음 완료 / MV 수령 직후 | 파트 재배정·휴식 후 속행 또는 MV 없이 강행 — PHASE 3 내 완결 |
| PHASE 4 (출시 준비) | 출시 확정 직전 | 연기 또는 강행 — PHASE 4 내 완결 |
| PHASE 5 (공개 후) | 성과 확인 직후 | 다음 루프 방향성 힌트로 반영 |
| PHASE 6 (후속 활동) | 활동 진행 중 | 일정 조정 또는 활동 축소 후 종료 — PHASE 6 내 완결 |

돌발 서사 상세 체크포인트 구조는 [`02_돌발_서사_개입.md`](./02_돌발_서사_개입.md)를 참조한다.

## 루프 내 플레이어 의사결정 목록

루프 한 사이클에서 플레이어가 내리는 핵심 결정의 전체 목록이다.
곡 제작(PHASE 1~3)은 수록곡마다 반복되므로, 해당 결정도 곡 수만큼 반복된다.

| 시점 | 결정 내용 | 성격 |
|------|-----------|------|
| PHASE 1 (곡마다 반복) | 작곡팀 선택 | 전략적 선택 (비용·실력·돌발확률) |
| PHASE 1 (곡마다 반복) | 작곡팀 제안 수락 여부 | 수락 / 거절 후 재선택 |
| PHASE 1 (곡마다 반복) | 다음 수록곡 추가 여부 | 앨범 규모 결정 |
| PHASE 2 (곡마다 반복) | 파트 블럭별 담당 멤버 배정 | 핵심 창작 결정 |
| PHASE 3 (곡마다 반복) | 녹음 전 연습량 조정 | 자원 배분 (에너지 관리) |
| PHASE 3 (곡마다 반복) | 상담 실시 여부 (징조 대응) | 리스크 관리 결정 |
| PHASE 3 (곡마다 반복) | 이 곡의 MV 출시 전 의뢰 여부 | 선택적 (비용·일정 vs 출시 시 밸류 상승) |
| PHASE 4 | 앨범 구성 결정 (타이틀곡, 수록곡 배치) | 전략적 선택 |
| PHASE 4 | 출시 방식 결정 | 음원 등록 vs 피지컬 발매 |
| PHASE 6 | 곡별 MV 후공개 의뢰 여부 | 선택적 (공개 후 화제성 / 팬덤 반응 활용) |
| PHASE 6 | 안무 의뢰 여부 및 안무팀 선택 | 선택적 (후속 활동 규모 결정) |
| PHASE 6 | 무대 활동 종류 선택 | 음방 / 팬미팅 / 콘서트 등 |

## 루프 반복에 따른 성장 구조

창작 루프는 단순 반복이 아니다. 루프를 거칠수록 다음 요소가 누적·변화한다.

| 요소 | 변화 방향 |
|------|-----------|
| 멤버 스탯 | 연습·경험을 통해 성장. 파트 소화 밸류가 높아짐 |
| 팬덤 | 출시 결과에 따라 규모·취향이 변화 |
| 인지도 | 공개된 곡·앨범의 누적으로 증가 |
| 협력자 풀 | 인지도 상승에 따라 더 좋은 작곡팀·안무팀 접근 가능 |
| 멤버 관계 | 창작 결정과 돌발 사건에 따라 친밀도·갈등도 변화 |
| 사장 평판 | 결정의 품질에 따라 업계 평판 변화 (추후 정의) |
