# 앨범 기획 및 출시

이 문서는 창작 루프의 PHASE 5~6에 해당하는 **앨범 기획 및 공개 과정**을 정의한다.
녹음완료 이후 앨범을 구성하고 공개하는 절차, 그리고 공개 후 성과 확인까지를 기술한다.

앨범 속성 정의는 [`01_창작_요소.md`](../02_개념/01_창작_요소.md)를 참조한다.

## PHASE 5: 앨범 기획 및 출시 준비

### 진입 조건
- 앨범에 담을 곡이 **녹음완료** 상태 이상이어야 한다
- 앨범에는 **하나 이상의 곡**이 필요하며, **하나 이상의 타이틀곡**이 있어야 한다

### 뮤직비디오 의뢰 (선택)

앨범 공개 전 타이틀곡에 뮤직비디오를 붙이면 밸류를 추가로 끌어올릴 수 있다.
의뢰는 **녹음완료** 상태 이후부터 가능하다.

```
뮤직비디오 의뢰 여부 결정
      │
      ├─ [생략] → 앨범 기획으로 바로 이동
      │
      └─ [의뢰]
            │
            ▼
      뮤직비디오 촬영팀 선택 (속성 확인)
            │
            ▼
      촬영팀이 제안 제시
        - 페이 (확정)
        - 컷 블럭 프리뷰 (장르, 등장 멤버 구성, 컷 밸류 범위 힌트)
            │
            ├─ [수락] → 페이 지불 → 대기 → 뮤직비디오 수령
            └─ [거절] → 다른 촬영팀 재탐색 또는 생략 결정
```

### 앨범 기획

플레이어가 직접 결정하는 항목이다.

| 항목 | 설명 |
|------|------|
| 앨범 이름 | 플레이어가 직접 입력 |
| 타이틀곡 선택 | 팬덤 반응에 가장 큰 영향을 미치는 대표곡 지정 |
| 수록곡 선택 | 타이틀곡 외 앨범에 포함할 곡 선택 |

타이틀곡은 복수 선택이 가능하다. 단, 퀄리티가 낮은 곡을 타이틀로 포함시키면 오히려 팬덤 반응에 역효과가 날 수 있다.

### 출시 방식 결정

| 방식 | 초기 비용 | 수익 | 인지도 효과 | 팬덤 효과 |
|------|-----------|------|-------------|-----------|
| 음원 등록 | 낮음 | 낮음 | 상승에 기여 | 미미 |
| 피지컬 앨범 발매 | 높음 | 높음 | 기여 | 팬덤 규모·몰입력 상승 |

- 두 방식은 **배타적이지 않다**. 음원 등록을 먼저 진행하고, 수익을 모은 후 피지컬 앨범을 별도로 출시하는 순차적 전략도 가능하다.
- 초기 자금이 부족하면 피지컬 앨범 선택이 불가할 수 있다
- 피지컬 앨범은 팬덤을 더 강하게 결속시키지만, 실패 시 손해도 크다

### 플레이어 판단 포인트
- 현재 만들어진 곡의 밸류 수준이 피지컬 앨범 발매를 정당화하는가
- 타이틀곡으로 선택한 곡이 팬덤 취향과 일치하는가
- 뮤직비디오 제작 비용을 투자할 여유가 있는가

## PHASE 6: 공개 및 성과 확인

### 앨범 공개

```
앨범 공개 실행
      │
      ▼
앨범 내 모든 곡 상태: → 공개완료
      │
      ▼
인지도 생성·증가 시작
팬덤 반응 산출
```

### 팬덤 반응 산출 구조

팬덤 반응은 다음 요소들의 조합으로 결정된다.

| 요소 | 영향 내용 |
|------|-----------|
| 앨범 최종 합산 밸류 | 결과물의 기본 품질. 높을수록 좋은 반응 |
| 타이틀곡 밸류 | 팬덤 반응에 가장 큰 비중을 차지 |
| 팬덤 취향 일치도 | 팬덤이 선호하는 장르·스타일과의 일치 여부 |
| 뮤직비디오 여부 | 있을 경우 인지도·팬덤 반응 추가 상승 |
| 멤버 퍼포먼스 완성도 | 안무숙지 여부, 멤버 소화 밸류 수준 |

### 취향 이탈

팬덤의 기존 취향과 다른 방향의 결과물이 나왔을 때 부정적 반응이 발생한다.

```
취향 이탈 발생 시
      │
      ├─ 인지도 상승폭 감소
      ├─ 팬덤 이탈 가능성 증가
      └─ 다음 루프에서 팬덤 취향 방향 재검토 필요
```

### 성과 확인 항목

공개 후 플레이어가 확인할 수 있는 정보이다.

| 항목 | 설명 |
|------|------|
| 인지도 수치 | 곡·앨범이 얼마나 알려졌는지 |
| 팬덤 반응 요약 | 긍정·부정 반응 지표 |
| 취향 이탈 여부 | 팬덤 취향과의 일치 방향 힌트 |
| 멤버별 주목도 | 어떤 멤버가 이번 활동으로 더 주목받았는지 |
| 수익 결산 | 이번 앨범으로 벌어들인 수익 |

## 루프 마무리 및 다음 루프 준비

공개 및 성과 확인이 끝나면, 플레이어는 다음 루프를 준비한다.

### 루프 종료 후 점검 사항

- **멤버 상태 점검**: 에너지·멘탈 회복이 필요한 멤버가 있는가
- **관계 변화 확인**: 이번 루프에서 멤버 간 친밀도·갈등도가 어떻게 변했는가
- **팬덤 취향 파악**: 이번 반응을 토대로 다음 루프의 방향성을 수정할 것인가
- **자금 확인**: 다음 작곡팀 의뢰에 필요한 자금이 충분한가

### 공개완료 이후에도 가능한 행동

공개완료 상태의 곡도 이후에 연습을 계속할 수 있다.
멤버 파트 소화 밸류를 꾸준히 높여두면 다음 루프에서 같은 장르 파트에 더 유리하게 작용한다.

| 행동 | 효과 |
|------|------|
| 노래연습 (공개완료 곡 기준) | 가창 스탯 장기적 성장 |
| 안무연습 (공개완료 곡 기준) | 안무 스탯 장기적 성장 |
