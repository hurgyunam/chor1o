# 돌발 서사 개입 구조

이 문서는 창작 루프의 각 단계에서 돌발 사건이 끼어드는 시점, 방식,
그리고 플레이어의 대응 흐름을 정의한다.

돌발 사건 유형 및 이야기꾼 정의는 [`03_돌발_서사_요소.md`](../02_개념/03_돌발_서사_요소.md)를 참조한다.

---

## 개입 원칙

- 돌발 사건은 각 PHASE 작업이 마무리되는 **체크포인트**에서 처리된다
- 체크포인트는 PHASE 흐름 안에 포함되어 있으며, PHASE를 역행하거나 건너뛰지 않는다
- 사건 발생 전에 **징조**가 먼저 나타나며, 플레이어는 탐색을 통해 이를 포착할 수 있다
- 플레이어가 징조를 무시하면 더 강한 사건이 발생한다
- 완벽한 예방은 불가능하다. 항상 일정 수준의 불확실성이 남는다

---

## PHASE 내 체크포인트 흐름

돌발 사건은 각 PHASE 작업이 끝나는 시점에 체크포인트로 처리된다.
체크포인트 내에서 해결이 완료되면 다음 PHASE로 진행한다.

```
PHASE 작업 완료
      │
      ▼
┌─────────────────────┐
│   돌발 체크포인트    │
└──────────┬──────────┘
           │
           ├─ [이야기꾼이 징조 발생]
           │         │
           │         ▼
           │   징조 출력 (멤버 침묵, 연습 불참 경고, 협력자 이상 신호 등)
           │         │
           │         ├─ [플레이어가 탐색하여 징조 포착]
           │         │         │
           │         │         ├─ [상담·재조정 실시] → 사건 예방 또는 약화
           │         │         └─ [무시]             → 사건 발생 (아래로)
           │         │
           │         └─ [플레이어가 징조 미포착] → 사건 발생 (아래로)
           │
           ├─ [이야기꾼이 즉시 사건 발생 결정 (징조 없이)] → 사건 발생 (아래로)
           │
           └─ [돌발 없음]
                     │
                     ▼
               다음 PHASE 진입

───────────────────────────────────────

사건 발생 시:
      │
      ▼
대응 선택 (PHASE 내에서 처리)
  ├─ [재조정]   → 현재 PHASE 내에서 계획 수정 후 확정
  ├─ [상담 실시] → 멘탈 회복. 상담 기간 해당 멤버 창작 참여 불가
  ├─ [휴식 지시] → 에너지 회복. 진행 속도 저하
  └─ [방치]     → 추가 사건 발생 또는 멤버 이탈 위험 증가
      │
      ▼
대응 결과 반영 → 다음 PHASE 진입
```

## PHASE별 돌발 개입 상세

### PHASE 1: 작곡 의뢰 중

| 사건 유형 | 설명 | 창작 루프 영향 |
|-----------|------|----------------|
| 작곡팀 계약 파기 | 수락한 작곡팀이 일방적으로 계약을 파기함 | 지불한 페이 일부 반환, 다른 작곡팀 재선택 필요 |
| 돌발 결과물 납품 | 기대 장르와 전혀 다른 곡이 납품됨 | 수락 시 파트 분배 계획 전면 재검토 필요 |

### PHASE 2: 안무 의뢰 중

| 사건 유형 | 설명 | 창작 루프 영향 |
|-----------|------|----------------|
| 안무팀 작업 지연 | 납기가 지연되어 전체 루프 일정이 밀림 | 출시 시점 지연 |
| 돌발 스타일 납품 | 의뢰 스타일과 다른 안무가 납품됨 | 파트 분배·포메이션 계획 재검토 필요 |

### PHASE 3: 파트 분배 중

| 사건 유형 | 설명 | 창작 루프 영향 |
|-----------|------|----------------|
| 멤버 파트 배정 불만 | 특정 멤버가 자신의 파트에 불만을 품고 징조가 발생함 | 방치 시 갈등 사건으로 악화. 파트 재조정 또는 상담 필요 |
| 멤버 간 갈등 심화 | 특정 멤버끼리의 갈등이 파트 배정 과정에서 드러남 | 포메이션 구성 제약, 상담 필요 |

### PHASE 4: 녹음 및 연습 중

| 사건 유형 | 설명 | 창작 루프 영향 |
|-----------|------|----------------|
| 멘탈 붕괴 | 특정 멤버가 심리적 한계에 달해 녹음 참여를 거부함 | 해당 멤버 파트 재배정 또는 상담 후 재시도 필요 |
| 멤버 부상 | 안무 연습 중 부상으로 참여 불가 | 파트 재배정 필수 |
| 과로 | 에너지 소모가 임계치를 초과해 퍼포먼스 저하 발생 | 녹음 결과물 품질 하락, 휴식 필요 |

### PHASE 5: 앨범 기획 및 출시 준비 중

| 사건 유형 | 설명 | 창작 루프 영향 |
|-----------|------|----------------|
| 루머·스캔들 | 멤버 관련 외부 루머가 확산됨 | 출시 시점 연기 또는 강행 선택 필요 |
| MV 촬영 사고 | 촬영팀 사고 또는 멤버 컨디션 저하로 촬영 중단 | MV 없이 출시 또는 재촬영 대기 |

### PHASE 6: 공개 후

| 사건 유형 | 설명 | 창작 루프 영향 |
|-----------|------|----------------|
| 취향 이탈 | 팬덤 취향과 다른 곡·안무 방향으로 부정적 반응 발생 | 다음 루프 방향성 재검토 필요 |
| 예상 밖 대박 | 기대 이상의 인지도 상승 | 다음 루프에서 더 높은 작곡팀·안무팀 접근 가능 |
| 이탈 위협 심화 | 공개 이후 주목받지 못한 멤버가 그룹 탈퇴를 언급 | 다음 루프 파트 분배 재검토 압박 |

## 상담과 창작 루프의 트레이드오프

상담은 돌발 사건의 핵심 대응 수단이지만, **창작 루프에 즉각적인 비용**을 발생시킨다.

| 상황 | 상담 선택 시 | 방치 선택 시 |
|------|-------------|-------------|
| 상담 기간 동안 | 해당 멤버 창작 참여 불가 → 파트 재배정 필요 | 창작 루프 계획 유지 가능 |
| 이후 결과 | 사건 예방·약화, 멘탈 회복, 스탯 정보 공개 | 더 큰 사건 발생 또는 퍼포먼스 품질 저하 |

이 트레이드오프가 돌발 서사가 창작 루프에 실질적인 영향을 미치는 핵심 경로다.
단순 텍스트 이벤트로 끝나지 않고, 플레이어가 **창작 계획 자체를 수정**해야 하는 상황을 만든다.

## 이야기꾼과 개입 빈도

이야기꾼 선택에 따라 돌발 개입의 빈도와 강도가 달라진다.

| 이야기꾼 | 루프 1회당 예상 개입 횟수 | 개입 강도 |
|----------|--------------------------|-----------|
| 매니저 지망생 | 1~2회 | 약~중 |
| 타블로이드 기자 | 3~5회 | 중~강 |
| 업계 베테랑 | 0~1회 | 강 (발생 시 큰 영향) |
