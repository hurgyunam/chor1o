# 창작 요소

이 문서는 플레이어가 직접 조작하는 핵심 창작 행위 요소들을 정의한다.
창작 요소는 이 게임의 **주축 재미**이며, 모든 시스템 설계의 중심이 된다.

---

## 1. 곡

케이팝 아이돌 그룹이 활동하기 위해 반드시 필요한 요소이다.

### 곡 속성

- 이름
- 곡 상태
- 곡 시간
- 다수의 파트 블럭
- 뮤직 비디오
- 최종 합산 밸류(인풋: 파트 블럭들의 최종 합산 밸류)
- 인지도
- 멤버별 에너지 소모량

#### 곡 상태의 순차적 변화


| 상태     | 다음 상태         | 플레이어 가능 행동     |
| ------ | ------------- | -------------- |
| 미완성    | 작곡완성          | 작곡의뢰           |
| 작곡완성   | 코레오완성, 파트분배완료 | 안무의뢰, 파트분배     |
| 코레오완성  | 파트분배완료        | 파트분배           |
| 파트분배완료 | 녹음완료          | 녹음, 노래연습       |
| 녹음완료   | 안무숙지완료, 공개완료  | 안무연습, 노래연습, 공개, 앨범기획 |
| 안무숙지완료 | 공개완료          | 노래연습, 공개, 앨범기획    |
| 공개완료   | X             | 안무연습, 노래연습 (멤버 파트 소화 밸류 향상 목적) |


#### 곡 상태에 따른 곡 속성 특징


| 속성          | 커스텀 가능 시기      | 생성 시기  | 확정 시기  | 발매 필수 조건 |
| ----------- | -------------- | ------ | ------ | -------- |
| 이름          | 미완성 ~ 공개완료     | X      | 공개완료 ~ | O        |
| 곡 상태        | 미완성 ~ 공개완료     | 미완성    | 공개완료 ~ | O        |
| 곡 시간        | X              | 작곡완성   | 작곡완성 ~ | O        |
| 다수의 파트 블럭   | 코레오완성 ~ 파트분배완료 | 작곡완성   | X      | O        |
| 뮤직 비디오      | 녹음완료 ~ 공개완료    | X      | 공개완료 ~ | X        |
| 최종 합산 밸류    | X              | 파트분배완료 | X      | X        |
| 인지도         | X              | 공개완료   | X      | X        |
| 멤버별 에너지 소모량 | X              | 파트분배완료 | X      | O        |


## 2. 파트 블럭

파트 블럭은 곡을 구성하는 최소 단위이다. 곡은 여러 파트 블럭의 조합으로 이루어진다.

### 파트 블럭 속성

- 가창 장르
- 최소/최대 요구 가창 레벨: 랩 스킬, 음정, 성량, 리듬감, 감성, 음색
- 곡 밸류
- 시작 시간(초), 끝 시간(초)
- 안무 장르
- 최소/최대 요구 안무 레벨: 파워, 리듬감, 정확도, 유연성, 표현력, 스태미나
- 안무 밸류
- 에너지 소모량
- 담당 멤버
- 멤버 소화 밸류
- 최종 합산 밸류(인풋: 곡 밸류, 안무 밸류, 멤버 소화 밸류)

#### 곡 상태에 따른 파트 블럭 속성 특징

| 속성       | 커스텀 가능 시기      | 생성 시기  | 확정 시기    |
| -------- | -------------- | ------ | -------- |
| 가창 장르    | X              | 작곡완성   | 작곡완성 ~   |
| 요구 가창 레벨 | X              | 작곡완성   | 작곡완성 ~   |
| 곡 밸류     | X              | 작곡완성   | 작곡완성 ~   |
| 시작 시간(초) | X              | 작곡완성   | 작곡완성 ~   |
| 끝 시간(초) | X              | 작곡완성   | 작곡완성 ~   |
| 안무 장르    | X              | 코레오완성  | 코레오완성 ~  |
| 요구 안무 레벨 | X              | 코레오완성  | 코레오완성 ~  |
| 안무 밸류    | X              | 코레오완성  | 코레오완성 ~  |
| 에너지 소모량  | X              | 파트분배완료 | X (멤버 소화 밸류에 연동되어 지속 변동) |
| 담당 멤버    | 코레오완성 ~ 파트분배완료 | 파트분배완료 | 파트분배완료 ~ |
| 멤버 소화 밸류 | X              | 파트분배완료 | X        |
| 최종 합산 밸류 | X              | 파트분배완료 | X        |

## 3. 작곡팀

곡을 만들기 위해 플레이어가 가장 먼저 의뢰해야할 팀이다.

협력자 속성 및 전략적 역할은 [외부 협력자 요소](./04_외부_협력자_요소.md)를 참조한다.

### 의뢰 가능한 곡 상태

- 미완성

### 작업 프로세스

- 작곡팀을 선정하고 곡 의뢰
- 작곡팀의 곡 제안 (페이, 파트 블럭 프리뷰)
- 곡 제안 수락시 페이 지불 및 대기 후 작곡된 곡을 받음

## 4. 안무팀

작곡완성된 곡의 밸류를 한층 더 끌어올려주기 위해 의뢰해야할 팀이다. 필수는 아니다.

협력자 속성 및 전략적 역할은 [외부 협력자 요소](./04_외부_협력자_요소.md)를 참조한다.

### 의뢰 가능한 곡 상태

- 작곡완성

### 작업 프로세스

- 안무팀을 선정하고 곡 의뢰
- 안무팀의 안무 제안 (페이, 파트 블럭 프리뷰)
- 안무 제안 수락시 페이 지불 및 대기 후 완성된 안무를 받음

## 5. 뮤직비디오

녹음완료된 곡의 밸류를 한층 더 끌어올려주는 요소이다.

### 뮤직비디오 속성

- 뮤직비디오 상태
- 다수의 컷 블럭
- 조회수


## 6. 뮤직비디오 컷 블럭

뮤직비디오 컷 블럭은 곡을 구성하는 최소 단위이다. 뮤직비디오는 여러 컷 블럭의 조합으로 이루어진다.

### 컷 블럭 속성

- 시작 시간(초), 끝 시간(초)
- 등장 멤버(0~전원)
- 장르
- 컷 밸류

#### 컷 블럭 장르

- 코레오
- 드라마틱
- 세계관
- 아트필름
- 시네마틱
- 애니메이션

## 7. 뮤직비디오 촬영 팀

녹음완료된 곡의 밸류를 한층 더 끌어올려주는 뮤직비디오를 만들기 위해 의뢰해야할 팀이다. 필수는 아니다.

협력자 속성 및 전략적 역할은 [외부 협력자 요소](./04_외부_협력자_요소.md)를 참조한다.

### 의뢰 가능한 곡 상태

- 녹음완료 ~

### 작업 프로세스

- 촬영팀을 선정하고 촬영 의뢰
- 촬영팀의 촬영 제안 (페이, 컷 블럭 프리뷰)
- 촬영 제안 수락시 페이 지불 및 대기 후 완성된 뮤직비디오를 받음

## 8. 앨범

곡 리스트. 

### 앨범 속성

- 이름
- 하나 이상의 곡
- 하나 이상의 타이틀 곡
- 최종 합산 밸류(인풋: 앨범 곡들의 최종 합산 밸류)
- 인지도
- 음원 등록 여부
- 피지컬 앨범 발매 여부

### 앨범 기획

플레이어가 해당 속성을 정함

- 이름
- 수록곡
- 타이틀곡

### 음원 등록

- 앨범의 곡들이 공개 상태로 변경
- 초기 비용이 적지만 수익도 적음
- 인지도 상승에 기여

### 피지컬 앨범 발매

- 앨범의 곡들이 공개 상태로 변경
- 초기 비용이 크지만 수익도 커짐
- 팬덤 규모 상승 및 몰입력 상승에 기여
