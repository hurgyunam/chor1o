# 02_개념

이 폴더는 게임에 들어갈 개념들을 먼저 정리하는 공간이다.
추후 완성될 스펙을 정하고 기능을 제한하는 데 기반이 되는 문서들을 담는다.
각 요소는 `01_재미요소/핵심_재미요소.md`의 판단 기준을 통과한 것들로만 구성된다.

---

## 파일 목록

### `01_창작_요소.md` — 곡 구조 / 파트 분배 / 안무 구성
플레이어가 직접 조작하는 핵심 창작 행위 요소들의 정의.

### `02_멤버_요소.md` — 멤버 스탯 / 특성 / 상담
그룹 멤버 개개인의 속성, 정보 공개 방식, 상담 시스템 정의.

### `03_돌발_서사_요소.md` — 돌발 사건 / 이야기꾼 / 난이도
창작 루프에 변수를 부여하는 돌발 서사 시스템 정의.

### `04_외부_협력자_요소.md` — 작곡가 / 안무가
플레이어가 고용·의뢰하는 외부 협력자 요소 정의.

### `05_메타_요소.md` — 게임 시작 설정 / 수익화 / UI 방향
게임 시스템 외부의 구조적 요소 정의 (게임 시작 선택, 수익화 모델, UI 설계 방향).
