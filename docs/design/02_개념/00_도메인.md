# 도메인 용어 정의

이 문서는 게임 설계 문서 전반에서 사용하는 핵심 용어들을 정의한다.
용어는 일관된 의미로 사용되어야 하며, 다른 문서에서 해당 용어가 등장할 경우 이 문서의 정의를 기준으로 한다.

---

## 인물

| 용어 | 정의 |
|------|------|
| **지원자** | 오디션에 참가하는 후보. 아직 소속사에 소속되지 않은 상태. |
| **연습생** | 오디션을 통해 선발되어 데뷔 전 육성 과정에 있는 사람. 아직 무대에 서지 않는다. |
| **멤버** | 아이돌 그룹으로 데뷔한 사람. 창작 루프의 핵심 자원이다. |
| **플레이어** | 소속사 사장 역할. 오디션, 창작 결정, 외부 협력자 선택 등 모든 주요 판단을 내린다. |

---

## 조직

| 용어 | 정의 |
|------|------|
| **아이돌 그룹** | 플레이어가 운영하는 케이팝 그룹. 멤버 여러 명으로 구성되며, 곡 활동의 주체가 된다. |
| **소속사** | 플레이어가 운영하는 조직. 그룹을 육성하고 앨범을 제작·출시한다. |

---

## 절차

| 용어 | 정의 |
|------|------|
| **오디션** | 지원자 중 적합한 인물을 선발해 연습생으로 데려오는 추리·판단 절차. |
| **데뷔** | 연습생이 처음으로 무대에 서는 시점. 이후부터 멤버로 호칭된다. |
| **상담** | 멤버의 심리 상태나 돌발 징조에 대응하기 위해 해당 멤버를 창작 활동에서 잠시 제외시키는 조치. |
| **창작 루프** | 곡 구조 설계 → 파트 분배 → 안무 구성 → 출시로 이어지는 반복 사이클. |

---

## 창작 개념

| 용어 | 정의 |
|------|------|
| **곡** | 한 번의 창작 루프에서 만들어지는 음악 결과물. 여러 파트 블럭으로 구성되며, 미완성 → 작곡완성 → 코레오완성 → 파트분배완료 → 녹음완료 → 안무숙지완료 → 공개완료 순서로 상태가 변화한다. |
| **파트 블럭** | 곡을 구성하는 최소 단위 구간. 각 블럭마다 가창 장르, 안무 장르, 담당 멤버가 할당되며, 블럭들의 최종 합산 밸류가 곡의 밸류를 결정한다. |
| **곡 밸류** | 파트 블럭의 가창 퀄리티를 나타내는 수치. 작곡팀이 결정한다. |
| **안무 밸류** | 파트 블럭의 안무 퀄리티를 나타내는 수치. 안무팀이 결정한다. |
| **멤버 소화 밸류** | 멤버가 해당 파트 블럭을 얼마나 잘 소화하는지를 나타내는 수치. 멤버의 능력치와 파트 블럭의 요구 레벨에 의해 결정된다. |
| **뮤직비디오** | 녹음완료된 곡의 밸류를 끌어올리는 선택적 요소. 여러 컷 블럭으로 구성된다. |
| **컷 블럭** | 뮤직비디오를 구성하는 최소 단위. 시간 구간, 등장 멤버, 장르, 컷 밸류 등의 속성을 가진다. |
| **앨범** | 타이틀곡과 수록곡을 묶어 출시하는 단위. 음원 등록 또는 피지컬 앨범 발매 방식으로 공개된다. |
| **타이틀곡** | 앨범의 대표 곡. 팬덤 반응에 가장 큰 영향을 미친다. |
| **수록곡** | 타이틀곡 외에 앨범에 포함되는 곡. 팬덤 취향 다양성에 기여한다. |
| **인지도** | 곡 또는 앨범이 공개된 이후 쌓이는 수치. 지속적으로 변동한다. |

---

## 역할

| 용어 | 정의 |
|------|------|
| **메인보컬** | 보컬 파트의 중심 담당. 코러스 및 감정 파트를 주로 맡는다. |
| **서브보컬** | 메인보컬을 보조하는 역할. 버스·프리코러스를 주로 담당한다. |
| **메인래퍼** | 랩 파트를 주로 담당하는 역할. |
| **서브래퍼** | 메인래퍼를 보조하는 랩 담당. |
| **센터** | 무대의 시각적 중심 역할. 포인트 안무와 카메라 노출이 집중된다. |
| **리드댄서** | 안무 포메이션의 선두를 이끄는 역할. |
| **메인댄서** | 전반적인 안무 퀄리티를 책임지는 핵심 역할. |
| **리더** | 멤버 결속과 사기 관리에 영향을 주는 선택적 역할. |

---

## 외부 협력자

| 용어 | 정의 |
|------|------|
| **작곡팀** | 플레이어가 의뢰하면 곡(파트 블럭 포함)을 제작해주는 외부 팀. 이름, 인지도, 평판, 과거 작업 밸류 등의 속성을 가지며, 의뢰 전 파트 블럭 프리뷰를 제안한다. |
| **안무팀** | 플레이어가 의뢰하면 파트 블럭에 안무를 추가해주는 외부 팀. 작곡완성 상태의 곡에만 의뢰 가능하며, 필수는 아니다. |
| **뮤직비디오 촬영팀** | 플레이어가 의뢰하면 뮤직비디오(컷 블럭 포함)를 제작해주는 외부 팀. 녹음완료 이후의 곡에만 의뢰 가능하며, 필수는 아니다. |
| **팀 제안** | 외부 협력팀이 의뢰 수락 전에 플레이어에게 제시하는 조건. 페이와 작업 결과물 프리뷰를 포함한다. |

---

## 서사 개념

| 용어 | 정의 |
|------|------|
| **돌발 사건** | 멤버 내부 또는 외부 환경에서 비롯된 예측 불가능한 사건. 창작 루프 외부에서 발생하며, 창작 루프 내 일부 선택지를 제한하는 방식으로 영향을 미친다. |
| **징조** | 돌발 사건이 발생하기 전에 탐색을 통해 발견할 수 있는 사전 신호. |
| **이야기꾼** | 돌발 사건의 빈도와 강도를 결정하는 난이도·성향 설정 요소. |
| **팬덤** | 그룹의 지지자 집단. 창작 결과물과 멤버 구성에 반응하며, 취향이 존재한다. |
| **취향 이탈** | 팬덤의 취향과 다른 방향의 결과물이 나왔을 때 발생하는 부정적 반응. |
