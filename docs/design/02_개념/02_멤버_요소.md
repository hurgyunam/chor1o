# 멤버 요소

이 문서는 그룹 멤버 개개인과 관련된 모든 요소를 정의한다.
멤버는 창작 행위의 대상이자 돌발 서사의 원천이다.

---

## 1. 멤버 구성

### 그룹 규모

- 그룹 멤버 수는 **4명 ~ 7명**으로 고정한다
- 그 이상으로 늘리려 할 경우 사장 캐릭터가 반대하는 서사로 자연스럽게 제한된다
- 멤버 수가 이 범위일 때 파트 분배·포메이션 결정의 복잡성과 각 멤버의 존재감이 동시에 살아있다

### 멤버 유형

| 유형 | 설명 |
|------|------|
| 연습생 | 데뷔 전 훈련 중인 상태. 스탯이 낮고 잠재력이 높다 |
| 정식 멤버 | 데뷔한 그룹의 활동 멤버 |

---

## 2. 멤버 스탯

### 가창 스탯

파트 블럭의 **요구 가창 레벨**과 직접 대응한다. 담당 멤버의 각 가창 스탯 수치가 파트 블럭의 최소~최대 요구 레벨 범위를 얼마나 충족하느냐에 따라 **멤버 소화 밸류**가 결정된다.

| 스탯 | 설명 |
|------|------|
| 랩스킬 | 랩 딕션과 플로우 실력. 랩 장르 파트에 활용 |
| 음정 | 음정 정확도와 음역대. R&B·발라드·댄스팝 파트에 활용 |
| 성량 | 목소리 크기와 울림. 정통 발라드 파트에 활용 |
| 리듬감 | 박자와 리듬을 타는 능력. 랩·댄스팝 파트에 활용 |
| 감성 | 감정 표현 능력. R&B·발라드·시티팝 파트에 활용 |
| 음색 | 목소리 고유의 색깔. 포크·시티팝 파트에 활용 |

### 안무 스탯

파트 블럭의 **요구 안무 레벨**과 직접 대응한다. 담당 멤버의 각 안무 스탯 수치가 파트 블럭의 최소~최대 요구 레벨 범위를 얼마나 충족하느냐에 따라 **멤버 소화 밸류**가 결정된다.

| 스탯 | 설명 |
|------|------|
| 파워 | 동작의 힘과 임팩트. 힙합·걸크러시 장르에 활용 |
| 리듬감 | 음악에 맞춘 안무 타이밍. 힙합·팝 장르에 활용 |
| 정확도 | 동작의 정밀도와 일치성. 걸크러시·팝 장르에 활용 |
| 유연성 | 부드럽고 넓은 동작 범위. 청순 장르에 활용 |
| 표현력 | 안무를 통한 감정 전달력. 청순·퍼포먼스 장르에 활용 |
| 스태미나 | 안무 지속 체력. 퍼포먼스 장르에 활용 |

### 기타 스탯

| 스탯 | 설명 |
|------|------|
| 비주얼 | 외모·무대 존재감. 센터 적합도에 영향 |
| 카리스마 | 무대 장악력. 타이틀곡 퍼포먼스 품질에 영향 |
| 멘탈 | 심리적 안정도. 돌발 사건 발생 가능성과 연결 |
| 체력 | 최대 에너지 한도에 영향. 체력이 낮을수록 에너지 소모량을 감당하기 어려워짐 |

### 에너지

스탯이 아닌 **소모 자원**이다.

- 파트 블럭마다 **에너지 소모량**이 존재하며, 곡의 파트 분배 완료 시 **멤버별 에너지 소모량**이 확정된다
- 멤버의 체력 스탯이 낮을수록 동일한 소모량에서 퍼포먼스 저하가 더 빨리 발생한다
- 에너지는 노래연습·안무연습·녹음 등의 활동으로 소모되며, 휴식으로 회복된다

### 스탯 공개 방식

- 스탯은 수치로 존재하지만 **처음부터 완전히 공개되지 않는다**
- 상담을 통해 점진적으로 수치를 파악할 수 있다
  - 상담량이 충분할 경우: 정확한 수치 공개
  - 상담량이 적을 경우: 근사치 또는 힌트 문구만 표출
- 힌트를 통해 추리하여 전략적 판단을 내리는 것이 의도된 플레이 방식이다
- 스탯은 시간 경과·훈련·사건에 따라 변화한다 (최신화 필요)

## 3. 멤버 특성 (성격 트레이트)

림월드의 성격 특성 시스템을 참조한다.

### 특성의 역할

- 각 멤버는 고유한 성격 특성을 보유한다
- 특성은 멤버의 행동 경향성과 돌발 사건의 원인이 된다
- 특성은 생성 시 부여되며, 일부는 활동 중 변화하거나 새로 생길 수 있다

### 특성 예시 (구체 수치는 추후 정의)

| 특성 예시 | 경향성 |
|-----------|--------|
| 완벽주의 | 결과물 품질에 집착. 동료와 갈등 유발 가능 |
| 소심함 | 돌발 사건에 더 크게 반응. 멘탈 스탯 회복이 느림 |
| 리더십 | 팀 사기를 높임. 갈등 중재 능력 보유 |
| 반항기 | 지시에 불응하거나 일탈 행동 가능성 있음 |
| 끼쟁이 | 무대 퍼포먼스 품질이 높지만 주목 경쟁 유발 가능 |
| 예민함 | 멤버 간 관계 변화에 민감하게 반응 |
| 성실함 | 스탯 성장 속도가 빠르고 돌발 사건 발생 가능성이 낮음 |

---

## 4. 멤버 간 관계

- 멤버 간에는 **친밀도**와 **갈등도**가 존재한다
- 관계가 창작 결정(파트 분배, 포메이션)에 따라 변화할 수 있다
- 관계 악화는 돌발 사건 발생 확률을 높이고, 관계 개선은 시너지 퍼포먼스를 가능하게 한다

---

## 5. 상담 시스템

### 목적

- 플레이어가 멤버의 심리 상태를 파악하고 사건을 예방하거나 수습하는 핵심 대응 수단이다

### 동작 방식

- 플레이어는 특정 멤버를 **상담에 전념**시키는 선택을 할 수 있다
- 상담 기간 동안 해당 멤버는 **창작 활동(파트 배분, 포메이션 등)에 참여할 수 없다**
- 상담을 통해 멘탈 회복, 갈등 해소, 스탯 수치 파악이 가능하다

### 상담의 트레이드오프

- 멤버 한 명이 상담에 빠지면 기존 창작 계획 전체를 재조정해야 한다
- 상담을 선택하지 않으면 더 큰 돌발 사건이 발생하거나 퍼포먼스 품질이 낮아진다
- 사전 상담(징조 발견 후 예방)과 사후 상담(사건 발생 후 수습) 모두 가능하다

### 상담과 정보 공개

- 상담 횟수와 깊이에 따라 멤버 스탯 수치가 점진적으로 공개된다
- 상담은 단순 이벤트가 아닌 **정보 탐색과 리스크 관리의 핵심 행위**다
