# 외부 협력자 요소

이 문서는 플레이어가 고용하거나 의뢰하는 외부 협력자 요소를 정의한다.
외부 협력자는 창작 루프에 **예측 불가 변수**를 추가하는 요소다.

의뢰 프로세스 및 곡 상태 연계는 [창작 요소](./01_창작_요소.md)를 참조한다.

---

## 1. 작곡가

### 역할

- 플레이어가 의뢰하면 곡을 제작해준다
- 플레이어는 사장으로서 작곡가를 직접 선별하여 고용한다

### 작곡가 속성

| 속성 | 설명 |
|------|------|
| 이름 | 작곡가 이름 |
| 인지도 | 업계 내 인지도 |
| 평판 | 작업 품질 및 태도에 대한 평판 |
| 최근 작업 아이돌 | 최근에 함께 작업한 아이돌 |
| 과거 작업 밸류 | 최근 TOP3, 최고 TOP3 |
| 장르 성향 | 해당 작곡가가 주로 작업하는 장르 (힙합, 발라드, 댄스팝 등) |
| 실력 | 결과물의 기본 품질에 영향 |
| 비용 | 의뢰 비용. 실력이 높을수록 비쌈 |
| 돌발 확률 | 기대 장르에서 벗어난 결과물이 나올 가능성 |

### 작곡가 제안

- 페이
- 파트 블럭 프리뷰

### 돌발 결과물

- 평소 힙합을 만들던 작곡가가 뜬금없이 발랄한 곡을 만들어내는 등, **의뢰 결과가 예상을 벗어날 수 있다**
- 이 경우 플레이어는 해당 곡을 타이틀곡으로 쓸지, 수록곡으로 남길지 전략적으로 선택해야 한다
- 돌발 결과물은 단순 실패가 아니며, 예상치 못한 히트를 낼 수도 있다

## 2. 안무가

### 역할

- 플레이어가 의뢰하면 안무를 제작해준다
- 플레이어는 사장으로서 안무가를 직접 선별하여 고용한다

### 안무가 속성

| 속성 | 설명 |
|------|------|
| 이름 | 안무가 이름 |
| 인지도 | 업계 내 인지도 |
| 평판 | 작업 품질 및 태도에 대한 평판 |
| 최근 작업 아이돌 | 최근에 함께 작업한 아이돌 |
| 과거 작업 밸류 | 최근 TOP3, 최고 TOP3 |
| 스타일 성향 | 해당 안무가가 주로 작업하는 안무 스타일 (칼군무, 개성 강조, 퍼포먼스형 등) |
| 실력 | 결과물의 기본 품질에 영향 |
| 비용 | 의뢰 비용. 실력이 높을수록 비쌈 |
| 돌발 확률 | 기대 스타일과 다른 결과물이 나올 가능성 |

### 안무가 제안

- 페이
- 파트 블럭 프리뷰

### 안무가의 전략적 역할

- 비싼 안무가를 고용해 완성한 안무가 멤버 구성과 맞지 않을 경우, 타이틀곡 배치 재검토가 필요하다
- 안무가의 결과물이 파트 분배·포메이션 결정에 영향을 줄 수 있다

## 3. 뮤직비디오 촬영팀

### 역할

- 플레이어가 의뢰하면 뮤직비디오를 제작해준다
- 플레이어는 사장으로서 촬영팀을 직접 선별하여 고용한다

### 뮤직비디오 촬영팀 속성

> 작곡가 / 안무가와 동일한 속성 구조를 공유한다.
> 추후 각 팀의 특성에 맞게 속성이 분화될 수 있으나, 일단 이 구조로 시작한다.

| 속성 | 설명 |
|------|------|
| 이름 | 촬영팀 이름 |
| 인지도 | 업계 내 인지도 |
| 평판 | 작업 품질 및 태도에 대한 평판 |
| 최근 작업 아이돌 | 최근에 함께 작업한 아이돌 |
| 과거 작업 밸류 | 최근 TOP3, 최고 TOP3 |

### 뮤직비디오 촬영팀 제안

- 페이
- 컷 블럭 프리뷰

---

## 4. 외부 협력자 선택의 의미

- 외부 협력자 선택은 **창작 루프 내의 전략적 결정**이다
- 비용 대비 기대 효과와 돌발 확률을 고려하여 선택해야 한다
- 협력자의 결과물은 플레이어가 완전히 통제할 수 없으므로, 항상 대비 계획이 필요하다
- 협력자 선택이 잘못될 경우 그 비용과 결과는 플레이어의 책임이다 (선택의 책임 유지)
